## Get ERC20-Token TotalSupply (aka MaxSupply) by ContractAddress
Returns the current amount of an ERC-20 token in circulation.

```json
https://api-testnet.bitlayer.org/scan/api?module=stats&action=tokensupply&contractaddress=0xe9e7cea3dedca5984780bafc599bd69add087d56
```
